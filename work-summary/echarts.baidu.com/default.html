美白走项目（APP项目）

  • 项目psd稿件切图
  • 前端页面开发
    1. 游戏邀请、下载、炫耀等H5界面开发
    2. 贺卡模板页、贺卡选择页、贺卡编辑页、贺卡预览页
    3. 福袋页、购买记录、关于我们、我的助走团、我的资金
    4. 尝试开发Hybird APP。首页、闪屏页、动画、参数、刷新、Webview

  • 重构精简代码、可读性更强
  • 解决禅道上的一些bug

项目总结


  • 必须拆分H5界面CSS、JS、HTML代码，精简代码
  • 使用阿里字体图标，3W+，减少图片能非常有效提高访问速度
  • 手动压缩图片、修改文件的路径，前端需要自动化
  • 基于页面开发，代码重复率高、复用性不强
  • 重复造轮子，虽然能深入研究，但是全部代码都自己写，影响开发效率

51代充项目（WEB项目）

  • 理解需求、参与设计师的早期设计
  • 前端页面开发
    1. 首页、登录注册、支付页面、个人中心、商品筛选等界面
    2. 搜索栏、登录注册、导航栏开发、about、certified
    3. 账单列表、充值中心、提现中心以及弹层效果
    4. 海外代购、游戏代充、苹果代充、话费代充、QQ代充、VIP代充
    5. 实现点击滚动到顶部功能、界面切换特效
  • 考虑移动优先、响应式布局设计；兼容IE7+、firefox、chrome、opera、safari等主流浏览器

  • 解决禅道上的一些bug：字体跨域请求问题，优化网站样式

项目总结


  • 前后端不分离，存在沟通成本、bug隐患、开发效率
  • 手动压缩图片、修改文件的路径，前端需要自动化
  • 基于页面开发，代码重复率高、复用性不强
  • 学习能力瓶颈，需要学习更强大的技术
