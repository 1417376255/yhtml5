## 五子棋

###  产品需求

* 使用原生技术实现，兼容 Chrome 浏览器即可。
* 实现胜负判断，并给出赢棋提示；任意玩家赢得棋局，锁定棋盘。
* 请尽可能的考虑游戏的扩展性，界面可以使用 DOM / Canvas 实现。考虑后续切换界面实现的方式成本最低。（比如选择使用 DOM 实现界面，需求* 为使用 Canvas 实现时尽可能少的改动代码）
* 实现一个悔棋功能
* 实现一个撤销悔棋功能。
* 人机对战部分可选。
* 尽可能的考虑实现的灵活性和扩展性。
