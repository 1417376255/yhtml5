## class name rules


### rules
* y-   -- uniquely
* -p   -- percent
* -n   -- negative


#### Attribute-oriented demo
* y-mt-60      -- {margin-top:60px}
* y-mt-p-60    -- {margin-top:60%}
* y-mt-pn-60   -- {margin-top:-60%}

* y-btn-lg
