Web推广页后台API

说明

本文档用于PC端商户推广管理页，该模块下的所有接口数据均在本文档中有所说明。
有关参数version（软件版本号）的说明如下：
1.1.1.20151008_alpha
1:主版本号，表示架构修改或者增加重大功能，根据项目需求决定是否需要修改
1.1：次版本号，表示已有功能的增强、小改动、bug的修复等，根据项目需求决定是否需要修改
20151008：修改日期，根据项目需求决定是否需要修改
alpha:希腊字母版本号，有5种base，alpha，beta，release
base表示只有基本界面，没有任何功能
alpha表示未经过或只是简单经过测试版
beta表示需要经过大规模测试版
release表示稳定发行版
